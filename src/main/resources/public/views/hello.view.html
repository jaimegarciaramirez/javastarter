BridgePhase Java Web Application Starter Project Blab

Getting off the ground fast and easy!

This project serves as a starting step to create a AngularJS web application with a Java Spring backend. The front-end is styled using the Pure CSS framework. Pure provides easy to use class definitions that require little-to-no forced markup.

This page is mapped by the HelloWorldController.hello() method to the view "home". This view is located under the /src/main/resources/public/views directory. All views should go under this directory. The configuration for these settings is under the /src/main/resources/application.properties file.

The menu below shows AngularJS routing using the AngularUI Router. Use the AngularUI Router for your routing needs, it offers more features than the default provided ng-route.
